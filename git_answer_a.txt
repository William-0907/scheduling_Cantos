List if Workflow using Git:

- git init
- git config --global user.name "your_username" / git config --local user.name "your_username"
- git config --global user.email "your_@username.com" / git config --local user.email "your_@username.com"
- git branch -M main
- create your file/folder/project
- git add *
- git commit -m "your commit message"
